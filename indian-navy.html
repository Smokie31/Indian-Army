The Indian Navy


Weapons

Ship launched ballistic missiles

Prithvi is a tactical surface-to-surface short-range ballistic missile (SRBM) developed by Defence Research and Development Organisation (DRDO) of India under the Integrated Guided Missile Development Program (IGMDP).

Description

  • Name : Prithvi
  • Type : Short-range ballistic missile
  • Warhead weight: 500 kg (1,100 lb) - 1,000 kg (2,200 lb)
  • Origin Country: India

Submarine-launched ballistic missiles

Sagarika, also known by the code names K-15 or B-05, is an Indian submarine-launched ballistic missile (SLBM) with a range of 750 kilometres (466 mi) that was designed for retaliatory nuclear strikes.

Description

  • Name : Sagarika
  • Type : Short-range submarine launched ballistic missile
  • Warhead: High explosives, strategic nuclear weapon
  • Origin Country: India

Cruise/anti-ship missiles

The BrahMos is a medium-range ramjet supersonic cruise missile that can be launched from submarine, ships, aircraft or land. It is notably one of the fastest supersonic cruise missiles in the world.

Description

  • Name : BrahMos
  • Type : Cruise/anti-ship missiles
  • Warhead: 200 kg (440 lb) conventional semi-armour-piercing and nuclear 300 kg (660 lb) (air-launched)
  • Origin Country: India, Russia

Surface to air missiles

Barak is an Israeli surface-to-air missile (SAM) designed to be used as a ship-borne point-defense missile system against aircraft, anti-ship missiles, and UAVs.

Description

  • Name : Barak
  • Type : Short range surface-to-air missile
  • Warhead: 22 kg (49 lb) blast fragmentation warhead
  • Origin Country: Israel
