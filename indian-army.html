Handguns

The Pistol Auto 9mm 1A also known as IOF 9mm pistol is a semi-automatic pistol manufactured by Rifle Factory Ishapore. It's a licensed copy of the Browning Hi-Power, made using tooling acquired from John Inglis and Company.

Description

  • Name : Pistol Auto 9mm 1A
  • Type : Semi-automatic pistol
  • Caliber: 9×19mm Parabellum
  • Origin Country: India

SMGs

The MP5 is a 9x19mm Parabellum submachine gun, developed in the 1960s by a team of engineers from the German small arms manufacturer Heckler & Koch GmbH (H&K) of Oberndorf am Neckar. There are over 100 variants and clones of the MP5.

Description

  • Name : Heckler & Koch MP5
  • Type : Submachine gun
  • Caliber: 9×19mm Parabellum
  • Origin Country: West Germany

Sniper Rifles

The Dragunov sniper rifle is a semi-automatic designated marksman rifle chambered in 7.62×54mmR and developed in the Soviet Union.

Description

  • Name : Dragunov sniper rifle
  • Type : Designated marksman rifle, Sniper rifle
  • Caliber: 7.62×54mmR
  • Origin Country: India,Soviet Union
